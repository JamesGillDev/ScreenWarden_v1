# Changelog

All notable changes to this project are documented in this file.

## [1.1.0] - 2026-02-25

### Added
- Voice command management workflow (add, edit, delete, reset) in the Settings UI.
- Additional screenshots documenting voice command edit states and dropdown behavior.

### Changed
- Settings persistence and README/release documentation for public publishing.
- Release packaging now supports versioned artifact naming for GitHub releases.

### Fixed
- Voice command and settings issues addressed during the v1.1.x iteration work.

## [1.0.0] - 2026-01-21

### Added
- Initial public release of ScreenWarden.
- Tray-first screenshot capture workflow.
- Capture mode selection (active window and monitor-under-cursor).
- Windows Speech Recognition support for optional voice-triggered actions.
